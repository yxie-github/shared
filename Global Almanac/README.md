# 全球黄历 · Global Almanac

一个自包含、离线可用的老黄历（通胜）网页应用：宜忌、黄道吉日判定、时辰吉凶、
干支纳音、星宿九星、吉神方位、彭祖百忌、节气等，并支持全球时区切换。

## 文件结构

```
全球黄历/
├── index.html          多文件版入口（推荐用来二次开发）
├── styles.css          全部样式
├── app.js              应用逻辑（渲染、时区、翻页）
├── lib/
│   ├── lunar.js        历法核心库 lunar-javascript（已内置，无需联网）
│   └── LICENSE-lunar-javascript.txt
└── 黄历_单文件版.html    把上面所有内容合并成的单文件版（双击即开）
```

## 运行方式（任选其一）

- **单文件版**：直接双击 `黄历_单文件版.html`，浏览器打开即用。
- **多文件版**：在 VS Code 中打开本文件夹，对 `index.html` 右键
  → *Open with Live Server*（推荐），或直接双击 `index.html`。

> 不需要 `npm install`，不需要联网——历法库已内置在 `lib/lunar.js`。
> 注意：多文件版用 `file://` 直接双击时，个别浏览器对本地 `<script src>` 有限制，
> 用 Live Server / 任意本地服务器打开最稳妥；单文件版无此问题。

## 功能

- **全球时区切换**：右上角下拉选择城市（北京/香港/东京/柏林/伦敦/纽约/悉尼等），
  黄历按所选城市的「当天」显示，时辰高亮按当地时间计算；默认跟随浏览器本地时区。
- **黄道吉日判定**：按传统「值神」体系——青龙/明堂/金匮/天德/玉堂/司命为黄道吉日，
  其余为黑道——右下角红印章给出结论。
- **宜 / 忌 双栏**，附冲煞、建除十二神、纳音五行、九星、星宿、月相、胎神占方、
  喜神/财神/福神方位、彭祖百忌、吉神宜趋、凶煞宜忌。
- **时辰宜忌**：12 时辰逐时标吉/凶，点击查看该时辰干支与宜忌，当前时辰自动高亮。
- **分区页签（所有屏幕常驻）**：顶部「日期 + 宜忌 + 黄道吉日」始终显示，下方密集信息分为
  **详情 / 神煞 / 时辰** 三个页签，点页签切换；页签不依赖媒体查询，在手机、平板、桌面及本地文件
  （含 Edge `edge://external-file`）下都能正常显示。顶部红条上的 3 个圆点也是同一组切换器，
  与页签、面板三者联动高亮。
- **术语释义（轻点提示）**：带虚线的词——宜忌项、值神、建除、纳音、九星、星宿、胎神、彭祖百忌、冲煞、
  黄道吉日印章，以及**吉神宜趋 / 凶煞宜忌里的每一个神煞名**（河魁、大时、咸池、血忌、往亡、复日、
  天恩、母仓……）——轻点即从底部弹出解释，点遮罩或按 Esc 关闭。词典在 `app.js` 的 `GLOSS` 对象，可自行增改。
- 日期切换：`‹ ›` 按钮、日期选择器、键盘 `←/→`；「今」回到今天。

## 二次开发提示

- 改样式 → `styles.css`（设计令牌在 `:root` 顶部）。
- 改逻辑 → `app.js`：`ZONES` 是城市/时区列表；`render(y,m,d)` 负责整页渲染；
  `SHI` 是十二时辰与小时区间。
- 历法 API 来自 lunar-javascript（`Solar` / `Lunar` 全局对象），
  文档见 https://6tail.cn/calendar/api.html

## 说明与许可

- 黄历宜忌属传统民俗内容，仅供参考，非现代天文/科学结论。
